# Local Android Mirror v1.0.1

Patch instalasi dan launcher Windows.

## Perbaikan

- Menambahkan panduan Smart App Control yang aman: verifikasi SHA-256, lalu
  Unblock ZIP sebelum Extract.
- Launcher sekarang memverifikasi versi backend sebelum membuka dashboard.
- Backend lama milik aplikasi yang tercatat di PID file dihentikan sebelum versi
  baru dijalankan, sehingga port `3030` tidak lagi memakai proses lama.
- Installer tetap tidak membutuhkan Administrator dan tidak mematikan antivirus.

## Instalasi

1. Verifikasi SHA-256 ZIP dengan `SHA256SUMS.txt`.
2. Klik kanan ZIP, pilih **Properties**, centang **Unblock**, lalu Apply.
3. Extract ZIP dan jalankan `Install-Local-Android-Mirror.cmd`.

Paket belum ditandatangani sertifikat code-signing komersial. Jangan mematikan
Smart App Control; gunakan langkah Unblock hanya setelah hash paket cocok.
