# Local Android Mirror v1.0.0

Rilis installer pertama.

## Fitur

- Dashboard lokal pada `127.0.0.1:3030`.
- Pair dan connect Android 11+ melalui Wireless Debugging.
- Dukungan USB Debugging tanpa pairing IP.
- Multi-device scrcpy dengan alias perangkat.
- Indikator koneksi USB, WiFi ADB, dan emulator.
- Start, stop, wake, dan disconnect WiFi.
- Clipboard sync untuk copy/paste; audio forwarding dinonaktifkan.
- Away/Sleep setelah 1 jam idle dan kebijakan 5 jam idle.
- Runtime Node.js, ADB, dan scrcpy disertakan.

## Privasi

- Tidak ada cloud streaming.
- Tidak ada fitur ADB pull, push, delete, install APK, atau custom shell.
- Pairing code disensor pada log command.
- Alias dan log tetap berada di `%LOCALAPPDATA%\TOOLS-MIRROR`.

## Instalasi

Extract ZIP, kemudian jalankan `Install-Local-Android-Mirror.cmd`.
