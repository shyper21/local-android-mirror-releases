# Local Android Mirror v1.1.0

Rilis fitur updater dan installer Windows tunggal.

## Fitur baru

- Tombol **Check Update** dan **Download & Install** pada dashboard.
- Update hanya berasal dari repository GitHub Release resmi, melalui HTTPS.
- Installer selalu diverifikasi dengan SHA-256 sebelum dijalankan.
- Judul jendela scrcpy memakai inisial perangkat yang disimpan pengguna.
- Installer kini berupa satu file `Local-Android-Mirror-Setup-v1.1.0.exe` dengan
  ikon aplikasi; pengguna tidak perlu extract ZIP.
- Auto-update memakai paket ZIP internal yang diverifikasi SHA-256, sehingga
  proses update tidak perlu menjalankan EXE unsigned.

## Keamanan

- Server tetap bind hanya ke `127.0.0.1:3030`.
- Tidak ada file HP yang di-pull, di-push, dihapus, atau dikirim ke GitHub.
- Pairing code tetap disensor dari log.
- Microsoft Defender scan pada installer final tidak menemukan ancaman.

## Catatan Windows

Installer belum ditandatangani sertifikat code-signing komersial. Smart App
Control masih dapat menahan EXE berdasarkan reputasi. Jangan mematikan antivirus
atau Smart App Control; verifikasi SHA-256 dari release resmi.
